# Семёнов Авиа — сайт

Сайт сельскохозяйственной авиации с базой заявок и админ-панелью.

## Запуск (разработка)

```bash
cp .env.example .env   # настройте ADMIN_PASSWORD
npm install
npm run dev
```

- Сайт: http://localhost:5173
- API: http://localhost:3001
- Админка: http://localhost:5173/admin

## Продакшен

```bash
npm run build
npm start
```

Сайт и API на одном порту (по умолчанию 3001).

## Админ-панель

- URL: `/admin`
- Пароль задаётся в `.env` → `ADMIN_PASSWORD`
- Просмотр, редактирование, удаление заявок
- Статусы: Новая, В работе, Выполнена, Отменена

## Настройка контента

`src/config/site.ts` — телефон, цены, тексты.

## База данных

SQLite — файл `data/leads.db` (создаётся автоматически).
